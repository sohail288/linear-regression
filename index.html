This chart was made using d3.js and jquery. I used several resources to learn how to make it, these are located in the footer

To use the interactive chart you just need to click on the graph at any point. This action will create a new point element. Then, just use the input buttons to clear or draw a regression line. In addition you can clear the chart and start fresh again.

Note that this uses arrow functions in javascript. Use new versions of chrome or firefox

Several things to do with this:

  • Let user dynamically change the scale
  • Allow multiple regression lines to be shown to highlight differences in additional data
  • Delete points!
